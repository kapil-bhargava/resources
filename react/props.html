React Props - Complete Beginner to Advanced Guide

Learn how to pass data between components using Props in React

üîç What are Props?

Props (short for "properties") are special read-only attributes used in React to pass data from a parent component to a child component. They help make components reusable and dynamic.

üì¶ How to Use Props


// ParentComponent.jsx
import ChildComponent from './ChildComponent';

function ParentComponent() {
  return <ChildComponent name="Alice" age={25} />;
}

// ChildComponent.jsx
function ChildComponent(props) {
  return (
    <div>
      <h3>Hello, {props.name}!</h3>
      <p>Age: {props.age}</p>
    </div>
  );
}

üéØ Destructuring Props

You can directly extract props using ES6 destructuring:


function ChildComponent({ name, age }) {
  return (
    <div>
      <h3>Hello, {name}!</h3>
      <p>Age: {age}</p>
    </div>
  );
}

üî¢ Types of Props

  • String: title="Welcome"
  • Number: count={10}
  • Boolean: isLoggedIn={true}
  • Array: items={[1, 2, 3]}
  • Object: user={{name: "Alice", age: 25}}
  • Function: onClick={handleClick}
  • JSX / children: <Component>JSX Here</Component>

üë∂ Using children Prop

The children prop allows components to wrap JSX inside them.


function Card({ children }) {
  return <div className="card">{children}</div>;
}

// Usage
<Card>
  <p>This is inside the card.</p>
</Card>

‚öôÔ∏è Setting Default Props

You can define default values for props in case none are passed.


function Greeting({ name = "Guest" }) {
  return <h2>Hello, {name}!</h2>;
}

üöß Prop Drilling

Prop drilling is when you pass data through many layers of components even if only the deepest component needs it. This can make your code harder to manage.

üí° Alternative: useContext + createContext

To avoid prop drilling, React provides the Context API.


// 1. Create context
const UserContext = React.createContext();

// 2. Provide value
function App() {
  return (
    <UserContext.Provider value={{ name: 'Eve' }}>
      <Profile />
    </UserContext.Provider>
  );
}

// 3. Use value
function Profile() {
  const user = React.useContext(UserContext);
  return <h3>Hello, {user.name}</h3>;
}

üìå Summary

  • Props are read-only attributes passed from parent to child.
  • Props make components flexible and dynamic.
  • Use destructuring to simplify syntax.
  • children prop allows nested JSX.
  • Default props prevent undefined errors.
  • Context API is useful for global data access.
