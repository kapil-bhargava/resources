Understanding useContext and createContext in React

Easy example with explanation and separate files

What is useContext?

useContext is a React Hook that allows you to access context values directly without passing props down manually through each component.

Why use Context API?

  • Avoids prop drilling (passing props through many levels)
  • Provides global state access
  • Efficient and cleaner state sharing

Why is it called API?
The Context API refers to a set of functions and components (like createContext, Provider, useContext) offered by React to manage global data.

Summary

  • createContext(): Creates context object
  • Provider: Supplies data to children
  • useContext(): Reads context data in any child

This helps you share data across components without manually passing props.
