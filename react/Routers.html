What is BrowserRouter?

BrowserRouter is a component provided by the React Router library. It uses the HTML5 History API to keep the UI in sync with the URL. This enables you to build single-page applications (SPAs) with navigation without full page reloads.

Why Use BrowserRouter?

  • Enables client-side routing for seamless navigation.
  • Improves performance by avoiding full page reloads.
  • Keeps the UI and URL synchronized.
  • Supports deep linking and bookmarking of SPA routes.

Installation Command

npm install react-router-dom

If you face dependency conflicts or want to reinstall forcefully, use:

npm install react-router-dom --force

What is --force?
It forces npm to fetch the latest package version and override any version conflicts or warnings, which is useful when package versions clash.

Routing Setup in index.js

// index.js
import React from 'react';
import ReactDOM from 'react-dom/client';
import { BrowserRouter, Routes, Route } from 'react-router-dom';

import App from './App';
import Home from './Home';
import About from './About';

const root = ReactDOM.createRoot(document.getElementById('root'));

root.render(
  <React.StrictMode>
    <BrowserRouter>
      <Routes>
        <Route path="/" element="<Home />" />
        <Route path="/about" element="<About />" />
        {/* If you want App component on some route, add here */}
        {/* <Route path="/app" element="<App />" /> */}
      </Routes>
    </BrowserRouter>
  </React.StrictMode>
);

Supporting Components

Home.js

function Home() {
  return <h2>Welcome to the Home Page</h2>;
}

export default Home;

About.js

function About() {
  return <h2>About Us Page</h2>;
}

export default About;

How to Navigate Between Routes

Use the Link component from react-router-dom:

import { Link } from 'react-router-dom';

function Navbar() {
  return (
    <nav>
      <Link to="/">Home</Link>
      <Link to="/about">About</Link>
    </nav>
  );
}

Conclusion

BrowserRouter is essential for handling dynamic routing in React applications. It improves user experience by providing seamless navigation and allows building complex single-page applications efficiently.
