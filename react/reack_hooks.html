What are Hooks in React ?

Hooks are special functions in React that let you "hook into" React features like state and lifecycle methods from functional components.

Why use Hooks?

  • ‚úÖ Simpler Code: No need to use classes ‚Äî you can write stateful logic in cleaner functional components.
  • ‚úÖ Code Reusability: Share logic across components with custom hooks.
  • ‚úÖ Better Readability: Easier to read and understand compared to class components.
  • ‚úÖ Modern Best Practice:
    • React now encourages the use of hooks in most cases.
Example : 
    
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0); // useState hook

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}
