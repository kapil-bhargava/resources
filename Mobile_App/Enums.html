Dart Enums

Understand what enums are, with clear examples and real-life meaning

1. What are Enums? (Programming Definition)

An enum (short for enumeration) is a special data type in Dart that lets you define a set of named, constant values. Itâ€™s useful when you have a fixed list of possible options for something.

Instead of using random numbers or strings, enums make your code easier to read and less error-prone by using meaningful names.

2. Real-World Example of Enums

Think about the days of the week: Monday, Tuesday, Wednesday, etc. These are a fixed list of possible values. You could write them down as an enum called Day. This way, you can use Day.Monday instead of typing strings like "Monday" that might be misspelled.

3. Important Terms

  • enum: A data type that has a limited, fixed set of constant values.
  • constant: A value that cannot change once set.
  • named values: Human-readable names for each possible value.

4. How to Use Enums in Dart

enum Day {
  Monday,
  Tuesday,
  Wednesday,
  Thursday,
  Friday,
  Saturday,
  Sunday
}

void main() {
  var today = Day.Monday;

  print('Today is: $today');  // Output: Today is: Day.Monday

  // Check if today is weekend
  if (today == Day.Saturday || today == Day.Sunday) {
    print('It\'s weekend! Time to relax.');
  } else {
    print('It\'s a weekday. Time to work.');
  }
}

Explanation:

  • Day enum lists all the days of the week as named values.
  • today stores the current day as one of those enum values.
  • We compare today with specific enum values to decide if it is a weekend or not.

5. Why Use Enums?

  • They make your code easier to understand by replacing unclear numbers or strings with meaningful names.
  • They help prevent errors caused by typos because you must use the exact enum names.
  • They keep the possible values limited and fixed, making your program safer and predictable.

6. Summary

Enums in Dart are like named boxes that hold a fixed set of constant values. You use them to write clean, safe, and understandable code when working with limited options, such as days of the week, directions, or statuses.
