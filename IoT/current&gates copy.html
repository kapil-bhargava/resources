AND Gate

Output:

Arduino/ESP32 Circuit:

Connect two push buttons to D2 & D3, LED to D13 with resistor. LED lights only if both buttons pressed.

Arduino Code:

const int buttonA = 2;
const int buttonB = 3;
const int led = 13;

void setup() {
  pinMode(buttonA, INPUT_PULLUP);
  pinMode(buttonB, INPUT_PULLUP);
  pinMode(led, OUTPUT);
}

void loop() {
  int a = !digitalRead(buttonA); // pressed = HIGH
  int b = !digitalRead(buttonB);
  digitalWrite(led, a & b);
}

NOT Gate

NAND Gate
