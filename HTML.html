Understanding CSS in Detail

1. Developed By and When

CSS (Cascading Style Sheets) was developed by Håkon Wium Lie in 1994 while he was working at CERN. CSS was created to style HTML documents and separate structure from presentation.

2. What is CSS?

CSS stands for Cascading Style Sheets. It describes the presentation of an HTML or XML document, including layout, colors, fonts, and spacing, allowing attractive and consistent designs.

3. Types of CSS

  • Inline CSS: Applied directly inside HTML tags.
  • Internal CSS: Written inside the <style> tag in HTML.
  • External CSS: Written in a separate .css file linked via <link>.

4. Use Cases of CSS

  • Layout Design: Using Flexbox, Grid, and positioning.
  • Styling Text: Fonts, sizes, weights, colors, spacing.
  • Responsive Design: Media queries for device adaptability.
  • Animations: Keyframes, transitions for visual effects.
  • Theming: Centralized color and style control.

5. What are Properties and Values?

A property defines the style aspect, and a value defines its behavior or appearance.

Example:

h1 { color: blue; font-size: 24px; }

Here, color and font-size are properties, while blue and 24px are values.

6. History of CSS

CSS was first proposed in 1994 and officially released in 1996 as CSS1. CSS2 followed in 1998, and CSS3 in 2001 brought modular design with Flexbox, Grid, Transitions, and more.

7. Implementation of CSS in HTML

  • Inline: Inside HTML elements using the style attribute.
  • Internal: Inside <style> tags in the head.
  • External: Linked .css files via <link> tag.

Example of Internal CSS:

<head>
  <style>
    body {
      background-color: #f4f4f4;
    }
  </style>
</head>
