Detailed Introduction to HTML

Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It structures content using elements represented by tags (e.g., <p>, <a>, <div>). HTML is not a programming language but a markup language that tells browsers how to display content.

History and Development of HTML

Invented by: Tim Berners-Lee
Year: 1991 (at CERN)
Purpose: To allow researchers to share documents easily over the Internet.

Versions of HTML

  • HTML 1.0 (1991): Basic tags like headings, paragraphs
  • HTML 2.0 (1995): Standardized existing tags
  • HTML 3.2 (1997): Added tables, scripting, applets
  • HTML 4.01 (1999): Introduced CSS support
  • XHTML 1.0 (2000): XML-based HTML
  • HTML5 (2014+): Multimedia, semantic tags, APIs

Additional Form-Related Tags

  • <fieldset> (Block): Group of related form fields
  • <legend> (Inline): Title for the <fieldset>
  • <meter> (Inline): Scalar measurement
  • <progress> (Block): Shows progress of a task
  • <textarea> (Block): Multi-line text input
  • <select> (Inline): Drop-down menu
  • <option> (Inline): Item in <select> or <datalist>
  • <datalist> (Block): Suggestion list for <input>

Summary

HTML is the foundation of the web. It consists of elements categorized into block and inline. Tags like <form>, <input>, <details>, and <select> enrich interaction and structure. HTML has evolved through several versions, with HTML5 being the latest and most powerful.
