ðŸ”¹ What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It lets you lay content out in rows and columns, and has many features that make building complex layouts straightforward.

Unlike Flexbox which is one-dimensional (either row or column), Grid allows you to work with both rows and columns simultaneously, making it perfect for creating entire page layouts.

Key Benefits:

  • Two-dimensional control: Manage both rows and columns
  • Responsive by design: Built-in responsive capabilities
  • Alignment control: Precise positioning of items
  • Overlapping elements: Layer items on top of each other
  • Gap control: Easy spacing between items

Advanced Grid Examples

1. Named Grid Lines

.grid-container { display: grid; grid-template-columns: [sidebar-start] 250px [sidebar-end main-start] 1fr [main-end]; grid-template-rows: [header-start] 60px [header-end content-start] 1fr [content-end]; } .header { grid-column: sidebar-start / main-end; grid-row: header-start / header-end; }

2. Grid Template Areas

.layout { display: grid; grid-template-areas: "header header header" "sidebar main aside" "footer footer footer"; grid-template-columns: 200px 1fr 150px; grid-template-rows: 60px 1fr 40px; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .aside { grid-area: aside; } .footer { grid-area: footer; }

3. Responsive Grid with Auto-Fit

.responsive-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; } /* Items automatically wrap to new rows when space is limited */

4. Overlapping Grid Items

.overlap-grid { display: grid; grid-template-columns: repeat(4, 1fr); grid-template-rows: repeat(3, 100px); } .item1 { grid-area: 1 / 1 / 3 / 3; /* Spans 2x2 */ z-index: 1; } .item2 { grid-area: 2 / 2 / 4 / 4; /* Overlaps with item1 */ z-index: 2; }

ðŸ”¹ Grid Units Explained

fr Fractional Unit

The fr unit represents a fraction of the available space in the grid container.

grid-template-columns: 1fr 2fr 1fr; /* First column: 25%, Second column: 50%, Third column: 25% */

auto Auto Sizing

The auto keyword sizes tracks based on their content.

grid-template-columns: auto 1fr auto; /* First and last columns size to content, middle column takes remaining space */

minmax() Function

Defines a size range greater than or equal to min and less than or equal to max.

grid-template-columns: repeat(3, minmax(100px, 1fr)); /* Each column is at least 100px but can grow equally */

repeat() Function

Repeats track definitions to avoid repetitive code.

grid-template-columns: repeat(4, 1fr); /* Same as: 1fr 1fr 1fr 1fr */ grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Creates responsive columns that wrap */

ðŸ”¹ Grid vs Flexbox: When to Use What?

Use CSS Grid when:

  • You need to control both rows and columns
  • Creating complex layouts (entire page layouts)
  • You want to overlap elements
  • Working with a known layout structure
  • You need precise control over item placement

Use Flexbox when:

  • You're working in one dimension (row or column)
  • Content size is unknown or dynamic
  • You want items to "flex" and fill available space
  • Creating navigation bars, button groups
  • Centering content is the primary goal

ðŸ”¹ Browser Support & Best Practices

Browser Support

CSS Grid is supported in all modern browsers. For older browsers, consider:

  • Using @supports queries for progressive enhancement
  • Providing flexbox or float fallbacks
  • Testing in target browsers
@supports (display: grid) { .layout { display: grid; grid-template-columns: repeat(3, 1fr); } } /* Fallback for older browsers */ .layout { display: flex; flex-wrap: wrap; }

Best Practices

  • Start simple: Begin with basic grids and add complexity gradually
  • Use semantic HTML: Grid enhances structure, doesn't replace it
  • Consider accessibility: Ensure logical tab order and screen reader compatibility
  • Test responsiveness: Use auto-fit, minmax(), and media queries
  • Name your lines: Use named grid lines for better maintainability
