JavaScript Assignments on Array Methods, Loops & Conditions

  1. Create an array of 10 numbers. Use a loop to print only the even numbers.
  2. Find the sum of all numbers in an array using a loop.
  3. Use the filter() method to get all numbers greater than 50 from an array.
  4. Use map() to create a new array where each number is multiplied by 2.
  5. Write a function that checks if an array contains a specific value using a loop.
  6. Sort an array of strings alphabetically using sort().
  7. Reverse an array of numbers using reverse().
  8. Use a loop to count how many numbers in an array are divisible by 5.
  9. Create an array of student marks and print "Pass" or "Fail" based on each mark (condition: pass if mark >= 40).
  10. Use the reduce() method to find the product of all numbers in an array.
  11. Merge two arrays into one using the spread operator and remove duplicates using Set.
  12. Find the maximum number in an array without using built-in methods.
  13. Use forEach() to print each element with its index.
  14. Write a program that counts how many times a specific value occurs in an array.
  15. Using some(), check if any number in the array is negative.
  16. Using every(), check if all numbers in an array are positive.
  17. Filter out all strings from an array that also contains numbers.
  18. Find the second largest number in an array using a loop.
  19. Use a loop to remove all occurrences of a specific number from an array.
  20. Create an array of objects representing students (name, age). Use filter() to get students above age 18.
