Bitwise Operators in JavaScript

What is a Bit?

A bit is the smallest unit of data in computing. It can either be 0 or 1.

All numbers in JavaScript are stored internally as 32-bit signed integers for bitwise operations.

List of Bitwise Operators

Operator Symbol Description
AND & Returns 1 if both bits are 1
OR | Returns 1 if either bit is 1
XOR ^ Returns 1 if bits are different
NOT ~ Inverts all the bits
Left Shift << Shifts bits to the left and fills 0s on the right
Signed Right Shift >> Shifts bits to the right and preserves the sign
Zero-Fill Right Shift >>> Shifts bits to the right and fills left bits with 0s
