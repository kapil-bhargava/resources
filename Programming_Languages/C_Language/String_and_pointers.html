Advanced C Programming Topics

1. Strings in C

Know about Strings

Strings in C are arrays of characters ending with a null character '\0'. You can declare and use strings with char arrays.

char name[20] = "John";
printf("Hello %s", name);

Use <string.h> for string functions like strlen(), strcpy(), etc.

3. Structures in C

Structures group multiple variables of different types into one unit.

struct Student {
  int id;
  char name[50];
};

struct Student s1 = {1, "Alice"};
printf("ID: %d, Name: %s", s1.id, s1.name);

Useful for representing real-world entities like students, books, etc.

4. File Handling in C

File handling allows you to read from and write to files. Use <stdio.h> functions like fopen(), fprintf(), fscanf(), and fclose().

FILE *fptr;
fptr = fopen("data.txt", "w");
fprintf(fptr, "Hello, file!");
fclose(fptr);

Modes: "r" (read), "w" (write), "a" (append), etc.
