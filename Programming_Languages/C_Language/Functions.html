Functions and Recursion in C

Functions in C

A function is a block of code that performs a specific task. Functions help us reuse code, divide logic into manageable parts, and improve readability.

Why Use Functions?

  • Improves code reusability
  • Makes programs easier to debug and maintain
  • Allows dividing complex problems into simpler blocks

Types of Functions:

  • Library Functions – Provided by C (like printf(), scanf())
  • User-defined Functions – Created by the programmer

Syntax of a Function:

return_type function_name(parameters) {
  // body of the function
}

Example: Simple Function

#include <stdio.h>

void greet() {
  printf("Hello, welcome to C programming!\n");
}

int main() {
  greet();  // Function call
  return 0;
}

Output: Hello, welcome to C programming!

Function with Return Value

int add(int a, int b) {
  return a + b;
}

int main() {
  int result = add(5, 3);
  printf("Sum is %d", result);
  return 0;
}

Output: Sum is 8

Parameters vs Arguments

  • Parameters are variables listed in the function declaration/definition.
  • Arguments are actual values passed to the function when it is called.

Example:

// Function with parameters
void displayMessage(char name[]) {
  printf("Hello %s!\n", name);
}

int main() {
  // "John" is the argument
  displayMessage("John");
  return 0;
}

Output: Hello John!

Types of Functions by Arguments and Return:

  • With arguments and return value: int add(int a, int b)
  • With arguments and no return: void greet(char name[])
  • No arguments but return value: int getValue()
  • No arguments and no return: void display()

Recursion in C

Recursion is the process in which a function calls itself either directly or indirectly to solve a problem.

Important Concepts:

  • Base Case: Condition to stop recursion
  • Recursive Case: The part where the function calls itself

Example: Factorial using Recursion

int factorial(int n) {
  if (n == 0)
    return 1; // Base case
  else
    return n * factorial(n - 1); // Recursive call
}

int main() {
  int result = factorial(5);
  printf("Factorial of 5 is %d", result);
  return 0;
}

Output: Factorial of 5 is 120

Advantages of Recursion:

  • Simplifies code for problems like factorial, Fibonacci, tree traversal

Disadvantages of Recursion:

  • Consumes more memory due to function calls
  • Can lead to stack overflow if not handled correctly
