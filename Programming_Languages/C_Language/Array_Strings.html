Arrays and Strings in C Programming

What is an Array?

Know more in Detail

An array is a collection of variables of the same type that are stored in contiguous memory locations. Think of it as a row of boxes, each box holding one value, and all boxes being of the same kind (type).

Arrays help you store and manage multiple values using a single variable name with an index to access each value.

Key Terms:

  • Element: An individual value stored in an array.
  • Index: Position number of an element in the array, starting from 0.
  • Contiguous Memory: Array elements are stored one after another in memory.

Example: Declaring and Using an Integer Array

int numbers[5] = {10, 20, 30, 40, 50};
printf("%d\n", numbers[2]);  // Outputs 30

Here, numbers is an array of 5 integers. We access the 3rd element using numbers[2] because indexes start at 0.

What is a String in C?

Know more in Detail

A string is a sequence of characters stored as an array of char elements in C. It is terminated with a special character called the null character represented as '\0'.

This null character marks the end of the string so functions know where the string stops.

Key Terms:

  • Character: A single letter, digit, or symbol stored in a char variable.
  • Null Character ('\0'): A special character indicating the end of a string.

Example: Declaring and Printing a String

char name[] = "Kapil";
printf("%s\n", name);  // Outputs Kapil

Here, name is a character array holding the characters 'K', 'a', 'p', 'i', 'l', and '\0' (end marker).

How to Work with Arrays and Strings?

  • Accessing elements: Use indexes starting at 0, like array[0] for the first element.
  • Modifying elements: Assign new values to specific indexes, e.g., numbers[1] = 25;
  • Looping through arrays: Use loops like for or while to process elements.
  • String functions: Use built-in functions like strlen(), strcpy(), strcmp() from string.h to work with strings.

Example: Looping through an Array

int i;
int numbers[5] = {10, 20, 30, 40, 50};
for (i = 0; i < 5; i++) {
    printf("Element %d: %d\n", i, numbers[i]);
}

Common Operations with Strings

  • strlen(str) – Returns the length of the string (excluding '\0').
  • strcpy(dest, src) – Copies the source string into the destination.
  • strcmp(str1, str2) – Compares two strings and returns 0 if they are equal.
  • strcat(dest, src) – Concatenates (appends) the source string to the destination string.
