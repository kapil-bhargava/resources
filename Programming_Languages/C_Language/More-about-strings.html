Strings in C Programming

What is a String in C?

In C, a string is a sequence of characters terminated by a null character \0. C does not have a dedicated string data type, so strings are represented using character arrays.

Declaring and Initializing Strings

char name[20];              // Declaration
char greet[] = "Hello";    // Initialization with string literal
char message[6] = {'H', 'e', 'l', 'l', 'o', '\0'}; // Manual initialization

Reading and Printing Strings

char name[50];
scanf("%s", name);  // Reads a single word
printf("%s", name);

// For reading a full line
fgets(name, 50, stdin);

Common String Functions (from <string.h>)

  • strlen(str): Returns length of string
  • strcpy(dest, src): Copies src to dest
  • strcat(dest, src): Appends src to dest
  • strcmp(str1, str2): Compares two strings
  • strupr(str): Converts to uppercase (non-standard)
  • strlwr(str): Converts to lowercase (non-standard)

Example:

#include <stdio.h>
#include <string.h>

int main() {
  char str1[20] = "Hello";
  char str2[20] = "World";
  strcat(str1, str2);
  printf("%s", str1);  // Output: HelloWorld
  return 0;
}

Important Notes

  • scanf("%s") stops reading on space.
  • fgets() is preferred for reading lines.
  • Always ensure space for \0 when declaring string arrays.

Use Cases of Strings

  • Reading user input
  • Storing and displaying messages
  • String comparisons and modifications
  • Parsing data formats like CSV, JSON
