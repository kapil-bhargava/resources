Python Strings

A string in Python is a sequence of characters enclosed within single quotes (' '), double quotes (" "), or triple quotes (''' ''' or """ """). Strings are immutable, meaning once created, their contents cannot be changed directly. Instead, any modification creates a new string.

Creating Strings


# Different ways of creating strings
str1 = 'Hello'
str2 = "World"
str3 = '''This is
a multi-line
string.'''

print(str1)   # Hello
print(str2)   # World
print(str3)   # Multi-line string

String Characteristics

  • Strings are sequences of characters.
  • They are immutable (cannot be modified in place).
  • They support indexing and slicing.
  • They support a wide range of built-in methods for manipulation.

String Indexing and Slicing


text = "Python"

# Indexing (0-based)
print(text[0])   # P
print(text[-1])  # n (last character)

# Slicing
print(text[0:4])  # Pyth
print(text[:3])   # Pyt
print(text[2:])   # thon

Common String Methods

Method Description Example
str.upper() Converts to uppercase "hello".upper() ‚Üí "HELLO"
str.lower() Converts to lowercase "HELLO".lower() ‚Üí "hello"
str.capitalize() Capitalizes the first letter "python".capitalize() ‚Üí "Python"
str.title() Capitalizes each word "python string methods".title()
str.strip() Removes whitespace from both ends " hello ".strip() ‚Üí "hello"
str.replace(a, b) Replaces substring a with b "hello world".replace("world", "Python")
str.split() Splits string into list "a,b,c".split(",") ‚Üí ['a','b','c']
str.join() Joins elements of a list into a string ",".join(["a","b","c"]) ‚Üí "a,b,c"
str.find() Returns index of substring "hello".find("e") ‚Üí 1
str.count() Counts occurrences of a substring "banana".count("a") ‚Üí 3
str.startswith() Checks if string starts with a prefix "Python".startswith("Py") ‚Üí True
str.endswith() Checks if string ends with a suffix "Python".endswith("on") ‚Üí True

String Formatting


name = "Alice"
age = 21

# Using f-strings (recommended)
print(f"My name is {name}, and I am {age} years old.")

# Using format()
print("My name is {}, and I am {} years old.".format(name, age))

# Using % formatting
print("My name is %s, and I am %d years old." % (name, age))

Escape Characters


text = "Hello\nWorld"   # New line
print(text)

path = "C:\\Users\\Admin"  # Backslash escape
print(path)

quote = "She said, \"Python is fun!\""
print(quote)

Use Cases of Strings

  • Storing textual information like names, emails, and addresses.
  • Processing natural language (NLP).
  • Working with file paths and configurations.
  • Building user interfaces and messages.
  • Data parsing (e.g., splitting CSV text).

# Example: Counting words in a sentence
sentence = "Python is easy and powerful"
words = sentence.split()
print("Word Count:", len(words))  # 4
