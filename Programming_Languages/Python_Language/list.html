Python List - Detailed Explanation

1. What is a List?

A list in Python is a collection data type that is:

  • Ordered – maintains insertion order.
  • Mutable – can be modified after creation.
  • Heterogeneous – can store different data types (int, string, float, etc.).
nums = [1, 2, 3]
mixed = [1, "hello", 3.14, [4, 5]]

2. Creating Lists

a = []                 # empty list
b = list()             # using constructor
c = [1, 2, 3]          # literal
d = list((1, 2, 3))    # from tuple
e = list(range(5))     # [0, 1, 2, 3, 4]

3. Accessing and Slicing

lst = [10, 20, 30, 40, 50]
print(lst[0])     # 10
print(lst[-1])    # 50
print(lst[1:4])   # [20, 30, 40]
print(lst[::2])   # [10, 30, 50]

4. Modifying Lists

Lists are mutable, so elements can be added, changed, or removed.

lst = [10, 20, 30]

lst[0] = 100             # change element
lst.append(40)           # add at end
lst.insert(1, 15)        # insert at index
lst.extend([50, 60])     # add multiple
lst.remove(20)           # remove by value
lst.pop()                # remove last
del lst[0]               # delete by index
lst.clear()              # empty the list

6. List Comprehensions

Concise way to build lists:

# Squares
squares = [x*x for x in range(5)]  # [0,1,4,9,16]

# Conditional
evens = [x for x in range(10) if x%2==0]  # [0,2,4,6,8]

# Flatten
matrix = [[1,2],[3,4]]
flat = [n for row in matrix for n in row]  # [1,2,3,4]

7. Performance Notes

  • Indexing and appending are O(1).
  • Insert/remove at front is O(n).
  • Membership test (x in lst) is O(n).
  • Use collections.deque if frequent front operations are needed.
