üîÅ What is Recursion?

Recursion is a technique used to solve problems by breaking them down into smaller subproblems of the same type.

Every recursive function must have a base case to avoid infinite loops.

üß† Recursion Example

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Output: 120

‚úÖ When to Use Recursion?

  • Problems that can be broken into similar subproblems
  • Tree traversal (e.g. file directories, family trees)
  • Mathematical problems like factorial, Fibonacci, etc.
