Python Variables

Understand how to store and manage data using variables in Python.

📂 What are Variables?

A variable in Python is a name given to a memory location that stores data. Python automatically assigns the type based on the value.

📌 Creating Variables

x = 5         # integer
y = "Hello"  # string
z = 3.14      # float

✅ Rules for Naming Variables

  • Must start with a letter or underscore (_)
  • Cannot begin with a number
  • Can only contain letters, numbers, and underscores
  • Case-sensitive: Age and age are different

🧠 Dynamic Typing

Python allows variables to change type dynamically:

x = 10
x = "Ten"  # now x is a string

🔍 Checking Variable Type

x = 25
print(type(x))

📅 Multiple Assignments

a, b, c = 1, 2, 3
x = y = z = "Same"
