Python Data Types

What are Data Types?

In Python, data types specify the type of data a variable can hold. They help Python understand how the data should be stored and operated on.

Basic Data Types

1. Integer (int)

Stores whole numbers.

age = 25

2. Float

Stores decimal numbers.

price = 99.99

3. String (str)

Stores a sequence of characters (text).

name = "Ritika"

4. Boolean (bool)

Stores True or False values.

is_active = True

5. List

Stores multiple values in a single variable.

fruits = ["apple", "banana", "mango"]

6. Dictionary (dict)

Stores data in key-value pairs.

student = {"name": "Aman", "age": 21, "present": True}

Real-Life Analogy

Think of variables as containers and data types as labels on those containers. It tells Python what kind of data is inside.

Container Label (Data Type) Example
Jar int 5
Measuring cup float 1.5
Envelope str "Sumit"
Switch bool True
