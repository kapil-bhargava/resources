üß† What is Syntax?

Syntax is the set of rules that define how Python code must be written and structured to be understood and executed by the interpreter.

üìå Key Basics of Python Syntax

1. Print Statement

Used to display output to the screen:

print("Hello, World!")

2. Comments

Single-line comments start with #:

# This is a comment

3. Indentation

Python uses indentation (4 spaces recommended) instead of braces to define blocks of code:

if True:
    print("This is indented")

4. Variables

No need to declare the type. Python is dynamically typed:

x = 10
y = "Hello"
print(x, y)

5. Data Types

Basic types include:

  • int - Integer numbers
  • float - Decimal numbers
  • str - Text
  • bool - True/False

6. Input from User

name = input("Enter your name: ")
print("Welcome", name)

7. Basic Arithmetic

a = 10
b = 3
print(a + b)
print(a - b)
print(a * b)
print(a / b)
