Arrays and Strings in C

Arrays

An array is a collection of elements of the same data type stored in contiguous memory locations.

Declaration and Initialization

int numbers[5];              // Declaration
int marks[5] = {90, 85, 75, 88, 92};  // Initialization

Accessing Elements

Array elements are accessed using indices starting from 0.

printf("First mark: %d", marks[0]);

Example: Print all array elements

#include <stdio.h>
int main() {
  int arr[3] = {10, 20, 30};
  for(int i = 0; i < 3; i++) {
    printf("%d ", arr[i]);
  }
  return 0;
}

Multidimensional Arrays

C also supports multidimensional arrays (e.g., 2D arrays).

int matrix[2][3] = {
  {1, 2, 3},
  {4, 5, 6}
};

Strings

A string in C is an array of characters ending with a null character '\0'.

Declaration and Initialization

char name[10] = "Alice"; // Automatically adds '\0'
char name[] = {'A','l','i','c','e','\0'}; // Manual

Reading and Printing Strings

char str[50];
scanf("%s", str);   // reads a single word
printf("%s", str);

String Functions

  • strlen(str) – Returns length
  • strcpy(dest, src) – Copies one string to another
  • strcat(s1, s2) – Concatenates s2 to s1
  • strcmp(s1, s2) – Compares strings

Example: Concatenate Strings

#include <stdio.h>
#include <string.h>

int main() {
  char first[20] = "Hello, ";
  char second[] = "World!";
  strcat(first, second);
  printf("%s", first);
  return 0;
}
