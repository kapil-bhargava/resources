Introduction to Java

Java is a widely used, platform-independent, object-oriented programming language. It is designed to have minimal implementation dependencies, allowing developers to write once and run anywhere (WORA).

Java Basics

Data Types

Java has two categories of data types: Primitive and Reference.

  • Primitive Types: byte, short, int, long, float, double, char, boolean
  • Reference Types: Objects, Arrays, Strings

Variables

Variables are used to store data values. They must be declared with a specific data type before use.

Operators

Operators perform operations on variables and values. Java supports arithmetic, relational, logical, bitwise, assignment, and ternary operators.

Control Statements

If-Else

Used for conditional branching based on boolean expressions.

Switch

Used for selecting one of many code blocks to execute.

Loops (for, while, do-while)

Loops execute a block of code multiple times until a condition is met.

Methods / Functions

Methods are reusable blocks of code that perform a specific task and can return a value.

Object-Oriented Programming (OOP)

Classes and Objects

A class is a blueprint for objects. An object is an instance of a class.

Inheritance

Inheritance allows one class to inherit the properties and methods of another.

Polymorphism

Allows methods to behave differently based on the object that calls them.

Abstraction

Hides complex implementation details and shows only essential features.

Encapsulation

Restricts direct access to some of an object's components and protects object integrity.

Exception Handling

Mechanism to handle runtime errors, using try, catch, finally, and throw.

Arrays

Containers that hold fixed-size collections of elements of the same type.

Strings

Immutable objects representing sequences of characters with many useful methods.

Collections Framework

List Interface

Ordered collections (arrays that can grow) like ArrayList, LinkedList.

Set Interface

Collections that do not allow duplicates, e.g., HashSet, TreeSet.

Map Interface

Stores key-value pairs, e.g., HashMap, TreeMap.

File Handling

Reading and writing files using classes like FileReader, FileWriter, and BufferedReader.

Input and Output (I/O)

Managing user input and program output using Scanner, System.in, System.out, and streams.

Multithreading & Concurrency

Threads Basics

Creating and running threads using Thread class or Runnable interface.

Synchronization

Control access to shared resources to avoid race conditions.

Java 8 and Later Features

Lambda Expressions

Concise way to represent anonymous functions.

Streams API

Functional-style operations on streams of elements, e.g., map, filter, reduce.

Optional Class

Container object to avoid null checks and NullPointerException.

Advanced Topics

Generics

Enables classes, interfaces, and methods to operate on typed parameters.

Annotations

Metadata to provide data about a program that is not part of the program itself.

Reflection

Allows inspection and modification of runtime behavior of applications.

JDBC (Java Database Connectivity)

JDBC is an API in Java that allows Java programs to connect and execute queries with databases. It provides classes and interfaces to interact with databases such as MySQL, Oracle, etc.

  • Steps in JDBC:
  • 1. Load the JDBC Driver
  • 2. Establish a Connection
  • 3. Create a Statement
  • 4. Execute the Query
  • 5. Process the Result
  • 6. Close the Connection

Common classes include: DriverManager, Connection, Statement, PreparedStatement, ResultSet.

Design Patterns

Reusable solutions to common problems in software design. Common Java design patterns include:

  • Creational: Singleton, Factory, Abstract Factory
  • Structural: Adapter, Decorator, Facade
  • Behavioral: Observer, Strategy, Iterator
