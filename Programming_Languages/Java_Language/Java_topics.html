Java Programming Language Guide

Introduction

Java is a popular, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is widely used for building cross-platform applications, mobile apps, and enterprise software.

Java Basics

Data Types: Java supports various data types including primitive types such as int, double, boolean, char and reference types like classes and arrays.

Variables: Variables are containers for storing data values.

// Declaring variables
int age = 25;
double price = 19.99;
boolean isJavaFun = true;
char grade = 'A';

Operators

Java provides arithmetic, relational, logical, bitwise, and assignment operators.

// Arithmetic operators example
int sum = 10 + 5; // 15
int product = 10 * 5; // 50

// Relational operators example
boolean isEqual = (10 == 5); // false

// Logical operators example
boolean result = (true && false); // false

Control Statements

Control flow is managed using conditional statements and loops.

// if-else statement
if(age > 18) {
  System.out.println("Adult");
} else {
  System.out.println("Minor");
}

// for loop
for(int i = 0; i < 5; i++) {
  System.out.println(i);
}

Methods

Methods (functions) contain reusable blocks of code.

// Method example
public int add(int a, int b) {
  return a + b;
}

// Calling the method
int result = add(10, 5);
System.out.println(result); // 15

Classes and Objects

Java is object-oriented. You create classes and instantiate objects from them.

// Class definition
public class Car {
  String color;
  int speed;

  void drive() {
    System.out.println("Driving at " + speed + " km/h");
  }
}

// Creating an object
Car myCar = new Car();
myCar.color = "Red";
myCar.speed = 60;
myCar.drive();

Inheritance

Inheritance allows one class to inherit fields and methods from another.

// Base class
class Animal {
  void sound() {
    System.out.println("Animal sound");
  }
}

// Derived class
class Dog extends Animal {
  void sound() {
    System.out.println("Bark");
  }
}

Dog d = new Dog();
d.sound(); // Output: Bark

Polymorphism

Polymorphism allows methods to behave differently based on the object that invokes them.


Animal a = new Dog();
a.sound(); // Calls Dog's sound method: Bark

Abstraction

Abstraction hides complexity by exposing only necessary details.


// Abstract class
abstract class Shape {
  abstract void draw();
}

// Subclass provides implementation
class Circle extends Shape {
  void draw() {
    System.out.println("Drawing Circle");
  }
}

Encapsulation

Encapsulation restricts direct access to some of an object's components and is achieved using access modifiers.


public class Person {
  private String name;

  public String getName() {
    return name;
  }
  public void setName(String newName) {
    name = newName;
  }
}

Exception Handling

Exceptions are errors that occur during program execution. Use try-catch to handle them.


try {
  int result = 10 / 0;
} catch(ArithmeticException e) {
  System.out.println("Cannot divide by zero");
}

Arrays

Arrays store multiple values of the same type in a single variable.


int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // 1

Strings

Strings are objects that represent sequences of characters.


String greeting = "Hello, Java!";
int length = greeting.length();
System.out.println(length); // 12

File Handling

Java can read/write files using classes like FileReader and FileWriter.


import java.io.FileWriter;
import java.io.IOException;

try {
  FileWriter writer = new FileWriter("output.txt");
  writer.write("Hello, File!");
  writer.close();
} catch(IOException e) {
  e.printStackTrace();
}

Input/Output

Use the Scanner class to read user input from the console.


import java.util.Scanner;

Scanner sc = new Scanner(System.in);
System.out.print("Enter your name: ");
String name = sc.nextLine();
System.out.println("Hello, " + name);

Collections Framework

Java Collections provide data structures like List, Set, and Map to store groups of objects.


import java.util.ArrayList;

ArrayList list = new ArrayList<>();
list.add("Apple");
list.add("Banana");
System.out.println(list.get(0)); // Apple

Threads (Multithreading)

Java supports multithreading to perform multiple tasks simultaneously.


class MyThread extends Thread {
  public void run() {
    System.out.println("Thread running");
  }
}

MyThread t = new MyThread();
t.start();
