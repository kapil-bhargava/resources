Navigator APIs Overview

API Description
navigator.userAgentReturns browser's user agent string
navigator.languageReturns browser language (e.g. en-US)
navigator.languagesArray of preferred languages
navigator.platformReturns the operating system platform
navigator.onLineChecks if browser is online
navigator.cookieEnabledReturns if cookies are enabled
navigator.hardwareConcurrencyNumber of logical CPU cores
navigator.deviceMemoryDevice memory in GB (approx.)
navigator.connectionNetwork type and speed info
navigator.geolocationAccess to location (with permission)
navigator.mediaDevicesAccess camera/mic with permission
navigator.clipboardClipboard read/write access
navigator.permissionsQuery API permissions (e.g. geo)
navigator.credentialsAccess saved credentials
navigator.storageCheck storage and persistence
navigator.bluetoothAccess Bluetooth devices
navigator.usbAccess USB devices
navigator.wakeLockPrevent screen from sleeping
navigator.getBattery()Battery info (level, charging)
navigator.sendBeacon()Send data without blocking unload
navigator.serviceWorkerRegister service workers for PWA
navigator.vibrate()Vibrate device (mobile)
navigator.xrAccess AR/VR WebXR devices
navigator.hidAccess Human Interface Devices
navigator.serialRead/write to serial ports
navigator.locksUse Web Locks API
navigator.presentationSend to external display
navigator.schedulingCooperative scheduling API
