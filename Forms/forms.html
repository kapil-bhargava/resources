HTML Forms Guide (with <select> and <option>)

1. What is an HTML Form?

An HTML form allows users to input data and submit it to a server for processing.

<form action="/submit" method="POST">
  <!-- form controls here -->
</form>

2. The <form> Tag and Attributes

<form action="URL" method="GET|POST">
  <!-- form elements -->
</form>
AttributeDescriptionExample
actionURL where form data is sentaction="/submit"
methodHTTP methodmethod="POST"
enctypeData encoding typeenctype="multipart/form-data"
targetWhere to display responsetarget="_blank"
autocompleteEnable/disable autocompleteautocomplete="off"
novalidateDisable validationnovalidate

4. <select> and <option> Tags

<select> defines a dropdown menu, <option> defines individual options.

<select name="country">
  <option value="india">India</option>
  <option value="usa">USA</option>
</select>

<select> Attributes

AttributeDescriptionExample
nameName of dropdownname="country"
multipleAllow multiple selectionmultiple
sizeVisible options countsize="4"
requiredMandatory selectionrequired

<option> Attributes

AttributeDescriptionExample
valueData sent to servervalue="india"
selectedPre-selectedselected
disabledUnselectabledisabled

5. Complete Example

<form action="/submit" method="POST">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" required>

  <label for="gender">Gender:</label>
  <select id="gender" name="gender" required>
    <option value="">--Select Gender--</option>
    <option value="male">Male</option>
    <option value="female">Female</option>
  </select>

  <label for="skills">Skills:</label>
  <select id="skills" name="skills[]" multiple size="3">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="javascript">JavaScript</option>
  </select>

  <button type="submit">Submit</button>
</form>
