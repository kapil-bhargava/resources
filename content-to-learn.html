🎨 Advanced CSS Techniques

  • :has() – Parent styling based on child state
  • clamp(), min(), max() – fluid typography/layout
  • CSS Grid – powerful 2D layout system
  • Scroll Snap & Scroll Timeline
  • CSS Variables – custom properties

🧠 DOM & JavaScript APIs

  • IntersectionObserver – detect visibility
  • MutationObserver – watch DOM changes
  • Drag & Drop API – native interactions
  • Debounce & Throttle – optimize performance

⚛️ React Tooling

  • Framer Motion – animations in React
  • React Query (TanStack) – data fetching
  • Zustand / Jotai – minimal global state
  • Tailwind Plugins – extend utility classes

🧩 Component Design Systems

  • Atomic Design Principles
  • Reusable UI libraries (buttons, modals, forms)
  • Accessibility-first (ARIA)
  • Headless UI design strategy

🏗️ Frontend Architecture

  • Folder & module structuring for scalability
  • Code Splitting with dynamic imports
  • Lazy Loading React components
  • Lighthouse for performance auditing

🛠️ Build Tools & Dev Setup

  • Vite or Parcel – modern bundlers
  • PostCSS – autoprefixing, nesting
  • Turborepo / Nx – monorepo tooling
  • TypeScript, ESLint – static type checking & linting

🚀 Practice Project Ideas

  • WhatsApp-style real-time chat app using Socket.IO
  • Scroll-snap image carousel with CSS-only animation
  • Dark/light theme toggle with localStorage
  • Admin dashboard with filters, charts, and responsive UI
